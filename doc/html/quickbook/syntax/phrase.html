Font Styles 

['italic], [*bold], [_underline], [^teletype], [-strikethrough]

will generate:

italic, bold, underline, teletype, strikethrough

Like all non-terminal phrase level elements, this can of course be nested: 

[*['bold-italic]]

will generate:

bold-italic
